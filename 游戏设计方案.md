## 一、游戏概述

### 1.1 基本信息

| 项目 | 说明 |
|------|------|
| **游戏名称** | 三维弹球 - 太空弹珠台 |
| **游戏类型** | 经典弹球街机游戏 |
| **游戏视角** | 3D 俯视视角（OrbitControls） |
| **核心玩法** | 使用挡板击打弹珠，撞击得分区获得分数，避免弹珠落入排水口 |
| **目标用户** | 休闲玩家、怀旧游戏爱好者 |

### 1.2 游戏特色

- **经典复刻**：还原 Windows 3D Pinball 经典玩法
- **3D 视觉**：Three.js 渲染，真实物理效果
- **连击系统**：2 秒内连续撞击，倍率递增（最高 5 倍）
- **彩色弹珠台**：红、黄、绿三色弹珠台，不同分数
- **实时物理**：Cannon-es 物理引擎，真实弹跳效果

## 二、游戏机制设计

### 2.1 核心规则

| 规则项 | 具体说明 |

| **生命系统** | 初始 3 条生命，弹珠落入排水口失去 1 条 |

### 6.2 音效参数

#### Web Audio API 参数示例

// 碰撞音效 - 根据弹珠台类型变化音调
const bumperFrequencies = {
  red: 440,    // A4
  yellow: 330, // E4
  green: 220   // A3
}

### 6.3 音效实现

```typescript
// 使用 Web Audio API 生成音效
function playBumperSound(bumperType: 'red' | 'yellow' | 'green') {
  const audioContext = new AudioContext()
  const oscillator = audioContext.createOscillator()
  const gainNode = audioContext.createGain()
  
  oscillator.connect(gainNode)
  gainNode.connect(audioContext.destination)
  
  const baseFreq = bumperType === 'red' ? 440 : 
                   bumperType === 'yellow' ? 330 : 220
  
  oscillator.frequency.setValueAtTime(baseFreq, audioContext.currentTime)
  oscillator.frequency.exponentialRampToValueAtTime(
    baseFreq * 2, 
    audioContext.currentTime + 0.1
  )
  
  gainNode.gain.setValueAtTime(0.3, audioContext.currentTime)
  gainNode.gain.exponentialRampToValueAtTime(0.01, audioContext.currentTime + 0.1)
  
  oscillator.start(audioContext.currentTime)
  oscillator.stop(audioContext.currentTime + 0.1)
}
```

## 七、游戏流程设计

### 7.1 游戏状态机

```
START → MENU → GAME → GAME_OVER → MENU
                ↓
              PAUSE
```

### 7.2 游戏流程

#### 1. 启动游戏
- 显示主菜单
- 玩家按下空格键开始游戏
- 初始化游戏状态（分数=0，生命=3，连击=0）

#### 2. 游戏进行中
- 弹珠位于底部，等待发射
- 玩家按下空格键发射弹珠
- 使用挡板控制弹珠
- 撞击弹珠台获得分数
- 连击获得倍率加成

#### 3. 失去生命
- 弹珠落入排水口（y < -4.5）
- 生命减 1
- 弹珠重置到底部
- 连击重置为 0

#### 4. 游戏结束
- 生命归零（lives <= 0）
- 显示最终得分
- 显示"PLAY AGAIN"按钮

#### 5. 重新开始
- 玩家点击按钮
- 重置分数、生命、连击
- 返回游戏状态
